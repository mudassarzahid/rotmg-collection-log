## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:mudassarzahid/rotmg-collection-log.git
   ```
2. Install required packages
   ```sh
   yarn install
   ```
3. Run the app
   ```sh
   yarn dev
   ```
